Il faut donner deux arguments aux programme pour que le filtrage par critère fonctionne.

En premier, le champ sur lequel le criètere va s'appliquer.
Il y a le choix entre 3 champs à filtrer :

* dept : filtre sur le département
* postal : filtre par code postal
* insee : filtre par code insee
